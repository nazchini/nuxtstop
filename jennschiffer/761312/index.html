Use another database!


Project structure allowing you to swap in another database

We structured these apps to handle the database operations in a dedicated module for the relevant provider. In the src folder youâ€™ll see sqlite.js, airtable.js, or astra.js, with methods called by the server script whenever an endpoint receives a request.
